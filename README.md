# Be The Hero

> Back-end: NodeJs
>
> Front-end: ReactJS 
>
> Mobile: React Native

### Descrição
Uma aplicação para Ongs registrarem casos e pessoas ajudarem com valores, podendo entrar em contato com as devidas ongs, ajudando-as e se tornando um herói!
